# DaVinci Resolve Studio Software — Professional Color Correction and Editing Environment

## About DaVinci Resolve Studio Software

DaVinci Resolve Studio Software provides a complete post-production environment for professional color correction, video editing, and visual effects. DaVinci Resolve Studio Software combines node-based color grading, magnetic timeline editing, and Fusion compositing. DaVinci Resolve Studio Software supports HDR grading with Dolby Vision and HDR10+ metadata. DaVinci Resolve Studio Software integrates Fairlight audio post-production directly into the timeline. DaVinci Resolve Studio Software enables users to collaborate simultaneously on the same project with cloud workflows. DaVinci Resolve Studio Software offers advanced tracking, magic mask, and Resolve FX for cinematic finishing. DaVinci Resolve Studio Software is trusted by Hollywood colorists, professional editors, visual effects artists, and broadcast finishing houses worldwide.
